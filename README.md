## Installation

1. Download the latest release
2. Upload to `/wp-content/plugins/`
3. Activate the plugin
4. Configure PDF URLs in post editor

## Configuration

Add PDF URLs in the post editor meta box. Buttons should have IDs:
- `view-pdf-btn` for View button
- `download-pdf-btn` for Download button

## Automatic Updates

This plugin supports automatic updates via GitHub.
